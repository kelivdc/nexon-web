# 🎨 NexonAce: Premium Design Strategy & Brand Identity

This document outlines the strategic design and technical implementation for **NexonAce**, a modern, global IT company.

---

## 🏛️ 1. Brand Identity & Positioning
NexonAce is positioned as the bridge between growth-stage businesses and world-class digital performance.

- **Primary Goal**: Transition from "Service Provider" to "Strategic Tech Partner."
- **Visual Language**: Silicon Valley / SaaS Aesthetic (Stripe/Apple-inspired).
- **Core Value**: Conversion-first mindset with 99.9% performance scores.

---

## 💎 2. Premium Design Tokens
The visual foundation is built on:
- **Electric Blue (#3b82f6)**: Trust, high-speed, and modern tech.
- **Glassmorphism**: Using `backdrop-blur` and translucent surfaces to create depth.
- **Layered Shadows**: Soft, multi-step shadows for a refined "card" feel.
- **Micro-Animations**: Subtle `rise-in` and hover scales to make the UI feel "alive."

## 🎓 4. CSR Focus: Free For Schools
Our **CSR Strategy** is integrated as a core brand feature:
- **Campaign**: "Digitalizing Education."
- **Offering**: School Profile, News Hub, Admissions.
- **Goal**: High trust and social proof while helping the community.

---

## 🛠️ 5. Technical Implementation Details
- **Framework**: TanStack Start (SSR/Hydration).
- **Styling**: Tailwind CSS v4 with custom `@theme` variables.
- **Components**: Functional React components with Lucide icons.
- **Optimization**: Core Web Vitals focused (fast LCP, no cumulative layout shift).
